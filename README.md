# UE5 Industrial Spline Simulation

A technical Unreal Engine 5 Blueprint-focused simulation prototype exploring spline-driven industrial vehicle systems, parametric movement behaviour, and modular simulation architecture.

## Project Goal

This project is focused on developing reusable Blueprint systems for industrial and infrastructure-style simulation workflows inside Unreal Engine 5.

The current prototype explores:

- Spline-driven maintenance vehicle movement
- Parametric speed control
- Smooth acceleration/deceleration systems
- Curve-aware movement behaviour
- Modular signal and marker systems
- Event-driven Blueprint architecture
- Debug visualization and simulation tooling

The project is being developed entirely in Blueprint as a technical simulation exercise.

---

##  Features

- Spline-following vehicle system
- Smooth speed interpolation
- Runtime spline traversal
- Dynamic rotation alignment
- Curve-based adaptive speed control
- Signal light system
- Stop markers and traffic logic
- Runtime track switching
- Debug UI tooling
- Multi-vehicle support

---

## Technical Focus

This project is intentionally focused on:

- Blueprint architecture
- Parametric systems
- Modular simulation logic
- Event-driven communication
- Reusable Unreal Engine systems

The goal is to create clean, scalable Blueprint systems suitable for industrial simulation workflows.
